# 项目简介

## 核心功能
- **用户认证中心**：实现全流程账号安全管理，包括用户注册、账号密码登录/安全登出、密码找回和老人专属快捷登录入口，确保账号安全。
- **前台用户端**
  - 首页门户：展示轮播图和热门公告，提供关键信息一目了然。
  - 智能监控板块：实时查看院内老人监护信息，支持图文/视频监控数据查看。
  - 社交互动专区：支持发布生活动态帖子，进行评论互动。
  - 公告信息中心：浏览系统全部公告，查看公告详情，实时接收院内通知。
  - 个人中心管理：查看与修改个人信息，管理个人发布内容。
  - 紧急远程求助：一键发布求助信息，实时对接管理员。
  - 个人业务办理：查看与修改健康档案，查询费用收缴明细，在线缴费支付。
- **管理员后端管理端**
  - 基础信息管理：管理老人基础信息，实现数据统一归档。
  - 智能监控管理：维护监控点位信息，支持图片和视频资源上传与管理。
  - 健康档案管理：管理老人健康数据全生命周期。
  - 财务费用管理：查看费用收缴明细，发布缴费通知，核对缴费状态。
  - 互动内容审核：审核院内交友帖子和评论。
  - 系统全局配置：管理首页轮播图素材，发布公告，查看与回复求助信息。

## 适用场景
适用于养老院的综合管理，满足老人、家属和管理员的多角色需求，覆盖从日常互动到紧急求助，从健康档案到财务管理的全场景运营管理。

## 技术特点
- 采用前后端分离标准架构，前端使用Vue3全家桶开发，打造高交互性、轻量化动态可视化界面。
- 后端基于SpringBoot3 + MyBatis开发，实现高效业务逻辑处理，支持高并发请求。
- 数据库适配MySQL 5.x/8.x版本，数据表设计规范化，确保数据安全与查询高效。
